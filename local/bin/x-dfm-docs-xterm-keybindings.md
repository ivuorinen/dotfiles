# x-dfm-docs-xterm-keybindings

Generate `docs/tmux-keybindings.md` using tmux's key list.

## Usage

```bash
x-dfm-docs-xterm-keybindings
```

No parameters are needed. The script writes the file under `docs/` and
overwrites any existing version.
