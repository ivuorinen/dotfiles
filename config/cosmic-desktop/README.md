# Catppuccin theme assets for COSMIC Desktop

Vendored verbatim from <https://github.com/catppuccin/cosmic-desktop>.

The COSMIC desktop applies these via its own GUI importers, not via
config-file symlinks, so the theme orchestrator does not flip them at
runtime — they live here so the import dialogs have a stable path.

- `themes/cosmic-term/catppuccin-<flavor>.ron` — terminal palettes.
  Apply via *COSMIC Terminal → View → Color schemes… → Import*.
- `themes/cosmic-settings/catppuccin-<flavor>-<accent>+round.ron` —
  desktop appearance presets. Apply via *COSMIC Settings → Desktop →
  Appearance → Import*.

Update policy: re-vendor with `git clone --depth=1
https://github.com/catppuccin/cosmic-desktop` and copy the `themes/`
subtrees over. Don't hand-edit — they're generated by upstream
`whiskers` from `cosmic-{term,settings}.tera`.
