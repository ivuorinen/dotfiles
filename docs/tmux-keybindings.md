# tmux keybindings

Leader: `<ctrl><space>`

Every registered binding in every key table, generated by `dfm helpers docs-tmux`
from `tmux list-keys`. Each entry shows the binding's note when it has
one (`bind-key -N`), otherwise the bound command. Built-in defaults are
included; mouse and wheel events are omitted.

## join-pane

```txt
"                      run-shell "'tmux-sessionist/scripts/join_pane.sh' 'join-pane' '-b' '-v'"
%                      run-shell "'tmux-sessionist/scripts/join_pane.sh' 'join-pane' '-b' '-h'"
-                      run-shell "'tmux-sessionist/scripts/join_pane.sh' 'join-pane' '-b' '-v'"
@                      run-shell "'tmux-sessionist/scripts/join_pane.sh' 'join-pane' '-b' '-b'"
f                      run-shell "'tmux-sessionist/scripts/join_pane.sh' 'join-pane' '-b' '-b'"
```
