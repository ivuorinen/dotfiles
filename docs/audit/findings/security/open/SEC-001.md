# grype reports 159 dependency CVEs in vendored / cached / gitignored paths

## Problem

grype reports 159 dependency CVEs, all in vendored, cached, or
gitignored paths the repo does not author or version-control.

## Evidence

All matches resolve under `config/vim/extra/fzf/**`,
`config/vim/plugged/**`, or `node_modules/**` — none of which is
tracked by git. Updates flow through the upstream tool chain
(yarn, vim-plug, fzf submodule sync).

## Impact

Informational only — vendored CVE exposure the repo does not
own. Accepted by maintainer ruling on 2026-05-05 (v1 status:
Accepted, informational — no repo-side fix).

## Fix

No repo-side fix. Re-running grype after a `yarn install` /
`vim +PlugUpdate` sweep will refresh the report.
